Battleship Game User Guide

About the program:

This program is a digitized version of the game Battleship. It is  a player vs PC type of game. There are 2 stages for the game,

  1. Setup Stage: Player puts the ships on the game board (on player side) at the location and rotation he wants; then the computer will also randomly place its ships on their side with different positions and rotation.
  2. Play Stage: Once the ships are settled, player and computer will take turns to attach each other’s area; Any hit on the ship location, the grid will show a red circle, otherwise it will show white circle. Whoever hits all the ship areas on the other side will win the game.
  3. At any time the player can reset the game and redeploy the ships.
  4. After the player / computer wins the game, it will ask for a rematch as well.

How to Play:

  1. “Setup” button, the program will go to “Setup Stage '', the ships are represented by circles, all circles will be placed in the center of the grid blocks chosen. The player will place different ships in a fixed order of lengths (e.x. 5 circles, 4 circles, 3, circles, etc.). The program will automatically move onto the next ship after one has been placed. If the player clicks on the PC’s side of the game board, an error message will pop up.
  2. “Reset” button: At any time the player can click the button to reset their selection of where they put their ships.
  3. After all the ships have been placed, the program will confirm with the player whether he is ready to play or not. If the player is not ready, they can choose “No” and the window closes and the player can reset the shipment layout. If the player is ready, the game will move onto the play phase of the game.
  4. During the play phase of the game, the player will not be allowed to attack their own side of the game board, only the PC’s side. The player and computer will take turns to hit a grid area on the other side. Red circles will pop up if a ship has been hit and white circles will pop up if the ships are missed. After either the player has hit all of the PC’s ships or the PC has hit all of the player’s ships, the program will tell the player whether they have won or lost and ask them whether they would like to play again. If they would like to, the game board is cleared and the game starts again. Otherwise, they exit the program.
  5. This user guide is also available when the user hits the “instructions” button.

Possible Error Messages:

  1. An error message that you may get is one that tells you you can’t place ships in that area. This error message should only pop up when the player clicks on a region that is outside of the expected region of where they’re supposed to be (in other words, out of their side of the game board) during the setup stage.
  2. If there is no icon on the popup window, please check if above icon files exist in the “icon” subfolder
  3. If there is no instruction, please check if the HelpDemo.html file is in the “help” subfolder.
  4. If the user is missing any used python packages, they need to use pip to install them.
